Cross-Language Code Smell Dataset and Detection Results
Overview

This repository contains a structured dataset of LLM-generated cross-language (Java + JNI/C/C++) code along with automated smell detection results.

The goal of this dataset is to support research and evaluation of cross-language code smells in systems that integrate multiple programming languages.

Repository Structure

The repository is organized into three main components:

1. Task Definitions (PDF)

A PDF file (Tasks for smells.pdf) describing all tasks used to generate code for each smell.

Defines the intent and requirements for each task
Covers all 16 cross-language smells
Each smell includes 5 distinct tasks

2. LLM generated codes

Contains code generated by multiple Large Language Models (LLMs).

Structure:
LLM generated codes/
│
├── LLM_1/
├── LLM_2/
├── LLM_3/
├── LLM_4/
    │
    ├── Smell_1/
    ├── Smell_2/
    ...
    ├── Smell_16/
         │
         ├── Task_1/
         ├── Task_2/
         ...
         ├── Task_5/
              │
              ├── *.java
              ├── *.c / *.cpp
Description:
4 LLMs are used
Each LLM generates code for:
16 smells
5 tasks per smell
Each task contains:
Java code
Native (C/C++) code using JNI
3. 📁 smell detection results on generated codes

Contains results of automated smell detection applied to the generated code.

Structure:
smell detection results on generated codes/
│
├── LLM_1/
├── LLM_2/
├── LLM_3/
├── LLM_4/
    │
    ├── Smell_1/
    ├── Smell_2/
    ...
    ├── Smell_16/
         │
         ├── Task_1.csv
         ├── Task_2.csv
         ...
         ├── Task_5.csv
Description:
Mirrors the structure of generated code
Each CSV file contains:
Detection results for one task
Information about whether the smell was detected
Relevant metrics or indicators (depending on the smell)
